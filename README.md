📦 Node Modules Not Included

The node_modules folder is not uploaded to this repository because it contains thousands of dependency files and can make the repository very large.

Instead, only the package.json and package-lock.json files are included. These files contain all the information needed to install the required dependencies.

🔧 How to Install Dependencies

After cloning the repository, run the following command:

npm install

This command will automatically download and install all required packages into the node_modules folder.
